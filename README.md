# Homepage

Uber simple placeholder html page form my private domain [zunh.dev](https://zunh.dev)

## Prequisites

- [Node](https://nodejs.org/en/) for runtime
- [pnpm](https://pnpm.io) for package management

## Run in development

```
$ pnpm i
$ pnpm run dev
```

## Build for production

```
$ pnpm i
$ pnpm run build
```
